## 项目名称 ：基于STM32F407的智慧农业大棚

#### 项目负责人 ：范志淋

#### 启动时间：2023.09.18

#### 项目语言及工具：

C语言，Keil5，Source Insight4.0，GY39传感器，TB6600步进电机驱动，28BYJ-48电机，HC-05蓝牙，OLED屏

#### 项目描述：

1.项目使用了两个USART串口分别对GY39模块采集的信息进行传输以及使用蓝牙模块进行通信控制系统。

2.项目利用IIC协议通信，将单片机收到的数据利用OLED屏实时显示

3.使用定时器输出脉冲控制TB6600步进电机驱动模块，利用定时器中断完成电机的4相电平切换运转电机，模拟风扇、灌溉

4.开启定时器14，持续调节PWM的占空比，模拟大棚补光灯效果

5.系统自动判断大棚当前环境，当温湿度超过设定值范围时控制蜂鸣器报警并驱动电机打开风扇和灌溉系统

6.配置PA4脚为ADC模拟功能，通过数模转换实时输出当前风扇和灌溉系统的工作电压U值

#### 项目功能：

版本：1.0

智慧大棚中，模拟出了温室大棚的环境。可以在智慧大棚中控制遮阳网、灌溉、补光灯、的开启和关闭，并且模拟农作物的生长过程。在接入GY39温度、湿度、光照传感器后，可以做到便捷快速的管理农业大棚。

在手机上即可通过蓝牙串口APP远程手动控制多个大棚的设施设备，包括风机、遮阳、喷滴灌、阀门、加温灯等。系统可按照设定时间，自动进行灌溉操作。
